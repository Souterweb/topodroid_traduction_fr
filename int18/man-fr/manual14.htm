[14] GESTIONNAIRE DE PROJETS TOPOGRAPHIQUES

Cette fenêtre est accessible à partir du bouton Projets de la Fenêtre principale.

Un projet de grotte regroupe les levés qui se trouvent dans la base de données de TopoDroid. Les "projets de grottes" sont stockés dans des fichiers de type Therion dans le dossier "thconfig" de TopoDroid. Un fichier de projet est sauvegardé chaque fois que le projet est fermé. Les projets peuvent être exportés sous forme de fichiers Therion ou Survex ; les exportations sont enregistrées dans les dossiers "th" et "svx", respectivement. Le gestionnaire de projet affiche la liste des projets cave dans le dossier "thconfig". La liste des projets est vide si vous n'avez pas encore créé de projet.

BOUTONS

  • Nouveau projet : ouvre une boîte de dialogue pour créer un nouveau projet de topographie.

MENU

  • FERMER : Ferme la fenêtre du gestionnaire de projet.
  • AIDE

En tapant sur le nom d'un projet, la Fenêtre du projet s'ouvre.
