MODIFIER LES COORDONNÉES D'UN POINT GÉOLOCALISÉ

Cette boîte de dialogue s'ouvre en appuyant brièvement sur un point de la liste des points géolocalisés.

La boîte de dialogue affiche :

  • la longitude, la latitude et l'altitude du géoïde
  • le nom et le commentaire du point
  • la déclinaison magnétique du point (à la date actuelle).


Boutons

  • Vue : montre le point avec une application spécifique (à condition de l'avoir installée)
  • Effacer les coordonnées converties. Cette action est immédiate.
  • Convertir les coordonnées de ce point dans un autre système de référence (en utilisant l'appli Proj4 si installée)
  • Supprimer l'enregistrement du point

Si les coordonnées du géopoint ont été saisies manuellement, elles peuvent être modifiées.

Les coordonnées prises avec le GNSS ou importées d'une application GNSS ne peuvent pas être modifiées. L'altitude du géoïde est modifiable, quelle que soit la méthode de saisie, si le paramètre approprié est défini.

Un appui long sur la longitude ou la latitude change l'affichage de DMS en degré et vice versa. L'altitude du géoïde est calculée à partir de l'altitude ellipsoïdale en utilisant le modèle de géoïde EGM2008.

Le bouton Convertir calcule les coordonnées du point dans un autre système de référence de coordonnées, en utilisant l''appli Proj4 (à condition de l'avoir installée). Les coordonnées converties sont affichées et stockées avec le point.

La déclinaison magnétique est calculée en utilisant le World Magnetic Model pour les années 2020-2025 (il est remplacé par un modèle WMM plus récent lorsqu'il est publié). L'incertitude de la déclinaison du modèle WMM peut atteindre 0,4 degré dans les régions tempérées.
Cochez la case de sauvegarder pour définir la déclinaison magnétique dans le relevé topographique.

Le bouton Enregistrer sauvegarde les modifications dans la base de données.
