EXPORTER UNE TOPOGRAPHIE FINALISÉE AU FORMAT Therion
("map" Therion)

Cette section décrit l'utilisation de Therion ou de cSurvey pour rédiger le fichier topographique final.

Therion

Si vous voulez compiler le projet Therion, vous devez fournir un fichier thconfig qui peut être aussi simple que ça :

source my_survey.th
export map -proj plan -o my_cave_p.pdf
export map -proj extended -o my_cave_s.pdf

Vous devez avoir exporté les données du relevé topo en tant que fichier Therion ".th", et les croquis en tant que fichiers Therion ".th2".
Le fichier de données contient des commandes pour saisir les fichiers de croquis. Cependant, elles sont commentées et vous devez les décommenter.
Le fichier de données contient des commandes pour définir une topo pour chaque croquis (scrap). Pour les topographies simples, vous pouvez laisser ces commandes en commentaire.Vous pouvez également exporter les croquis sous forme de fichiers Therion ".xvi" et les retracer à l'aide de xtherion.

cSurvey

Sinon, vous pouvez travailler avec cSurvey.
Ouvrez le fichier croquis exporté csx avec cSurvey : vous obtiendrez la feuille de données remplie avec les données topographiques et les croquis correctement alignés sur la page de dessin de cSurvey.
