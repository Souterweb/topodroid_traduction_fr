LISTE DES POINTS GÉOLOCALISÉS
Boîte de dialogue

Cette boîte de dialogue s'ouvre en appuyant sur le bouton GNSS (GPS) dans la Fenêtre d'Info du relevé topographique.

La boîte de dialogue affiche la liste des points, et dispose de trois boutons pour créer un nouveau point :

  • GPS : obtenir la position du point à partir d'un GNSS
  • Ajouter : permet d'entrer les coordonnées manuellement
  • Importer : importer une position à partir d'une application GNSS
  • Démarrer l'application GPSTest

Attention. La précision d'un point géolocalisé obtenu avec un téléphone est généralement de 10 m au mieux.

En appuyant brièvement sur une entrée de la liste on ouvre la boîte de dialogue Modifier les coordonnées du point.

En interne TopoDroid stocke les coordonnées en degrés décimaux. Mais dans les boites de dialogues de coordonnées vous pouvez les saisir soit en degrés décimaux soit sous la forme "jj:mm:ss.ss". Le clavier de TopoDroid possède des boutons spéciaux pour "degré" et "minute". Si votre clavier qui n'a pas de point-virgule (;), utilisez l'espace à la place. Si le clavier n'a pas le point décimal (.) utilisez la barre oblique slash (/) à la place.

Menus :

  • Fermer la fenêtre
  • Faire défiler les applications GNSS externes. S'il n'y a qu'une seule application GNSS, ce menu n'a aucun effet.
  • Aide


Applications GNSS externes

TopoDroid peut importer des coordonnées de géopoints à partir de :

  • MobileTopographer. Une boîte de dialogue s'ouvre pour choisir le point géolocalisé.
  • Fichier de trace de GPX RecorderLe dernier point de trace est utilisé.
  • GPS Position (csv). TopoDroid calcule une moyenne des positions.
  • GPSTest à partir d'un fichier log. TopoDroid utilise le dernier point de repère.
  • Fichier de trace GPS Logger sauvegardé au format "csv". TopoDroid utilise le dernier point de trace.


Le nom de la station d'un géopoint importé depuis GPX Recorder, GPX Position, et GPSTest est défini automatiquement par TopoDroid sous la forme d'un nombre préfixé par '#'.

Les fichiers de listes de points de MobileTopographer sont stockés dans le dossier MobileTopographer/pointlists.

Les fichiers de suivi de GPS Recorder sont stockés dans le dossier gpx_recorder.
