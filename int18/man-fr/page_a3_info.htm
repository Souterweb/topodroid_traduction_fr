INFORMATIONS SUR LE DISTOX1 A3

Cette boîte de dialogue est ouverte en appuyant sur le bouton Info de la fenêtre Périphérique.

Informations DistoX1 - A3 :

  • code de l'appareil
  • unités d'angle
  • boussole et clino on / off
  • mode normal / étalonnage
  • mode silencieux on / off

La boîte de dialogue comporte un bouton qui permet d'effacer la mémoire du DistoX, c'est-à-dire d'effacer le hot-bit qui marque les données à transmettre.
Après l'effacement, le DistoX doit être éteint et rallumé pour que l'action prenne effet.
