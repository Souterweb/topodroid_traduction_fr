IMPORT / EXPORT D'ÉTALONNAGE DISTOX

Importation d'étalonnage CSV

Un étalonnage peut être importé à partir d'un fichier d'étalonnage CSV.
Cette boîte de dialogue s'ouvre en appuyant sur Importer dans la boîte de dialogue de Liste d'étalonnage.
Cette boîte de dialogue affiche la liste des fichiers CSV d'étalonnage.

Appuyez sur l'entrée que vous souhaitez importer.

L'importation échoue si le fichier ne contient pas d'étalonnage, ou si l'étalonnage est déjà présent.
Elle peut également échouer si l'adresse Bluetooth de l'appareil inscrite dans l'étalonnage est différente de celle de l'appareil actuel.

Exportation d'étalonnage CSV

Un étalonnage de DistoX peut être exporté au format CSV (texte). La boîte de dialogue d'exportation s'ouvre à partir du menu "Export" de la Fenêtre d’étalonnage.
Ce format d'exportation contient les informations d'étalonnage : nom, date, adresse Bluetooth du périphérique (DistoX), commentaire, algorithme; suivies par les données d'étalonnage proprement dites et des coefficients d'étalonnage.

Chaque ligne de données contient :

  • Indexation des données
  • Les valeurs G et M (six entiers)
  • Groupe
  • Azimut, pente et rotation
  • Erreur
  • Statuts
