ÉDITION / MODIFICATION D'UNE ÉTIQUETTE D'ÉLÉMENT DU CROQUIS
Boîte de dialogue

Cette boîte de dialogue s'ouvre lorsque vous insérez une étiquette (en mode dessin de la Fenêtre de Croquis).

Propriétés de l'étiquette :

  • Taille : au choix parmi XS, S, M, L, XL
  • Orientation
  • Texte
  • Calques : les vues du fond sur lesquelles l'étiquette est affichée [T].
  • Options supplémentaires Therion


Les vues du fond sont présentées sous forme de cases à cocher avec le code alphabétique de la vue. Par défaut, les étiquettes appartiennent aux vues B "base" et A "artefact". Il est possible de désactiver certaines vues.
Attention, si l'étiquette n'est pas assignée à une vue de fond sélectionnée, elle n'apparaîtra pas sur le fond même si elle est effectivement ajoutée au croquis.
