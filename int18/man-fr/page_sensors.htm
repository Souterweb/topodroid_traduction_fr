CAPTEURS

Cette boîte de dialogue est ouverte en appuyant sur le bouton Capteur dans la boîte de dialogue d'Édition secondaire.

Elle contient une liste de cases à cocher pour sélectionner un capteur intégré, ou un capteur externe (dans ce cas, vous devez entrer le "type de capteur").

Elle contient également trois champs de texte :

  • type de capteur. L'un des capteurs intégrés répertoriés, ou un capteur à définir ici si vous sélectionnez un capteur "externe"
  • valeurs mises à jour automatiquement pour les capteurs intégrés
  • commentaire

Le bouton Enregistrer stocke la mesure dans la base de données.

Le bouton Android Retour ferme la boîte de dialogue sans enregistrer.
