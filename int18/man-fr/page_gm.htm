DONNEES D'ÉTALONNAGE DISTOX

Cette boîte de dialogue s'ouvre en appuyant sur une saisie de données dans la Fenêtre des données d'étalonnage.

  • Azimut, inclinaison, rotation
  • Erreur d'étalonnage
  • Champ de saisie du numéro de groupe
  • Enregistrer les modifications

Les boutons sont les suivants :

  • Regrouper : réaffecter les groupes à partir de ces données
  • Marquer les données comme supprimées

Le numéro de groupe de données peut être défini ou modifié.

  • Positif : les données prennent part au calcul du coefficient d'étalonnage
  • Zéro : les données ne contribuent pas au calcul

Les données avec le même numéro de groupe positif forment un groupe de données.

Les données supprimées ne sont pas retirées des données de la base de données.
Elles n'apparaissent généralement pas dans la liste des données d'étalonnage, ne sont pas prises en compte dans l'affectation automatique des groupes et n'entrent pas dans le calcul des coefficients d'étalonnage.
