[5] DONNÉES D'ÉTALONNAGE DU DISTOX

Valable uniquement pour les DistoX, DistoX2 et DistoXBLE

La Fenêtre d'étalonnage affiche la liste des données de d'étalonnage. Si la base d'étalonnage ne contient aucune donnée, la liste est vide, mais les boutons du haut sont disponibles. Chaque ligne de données contient les informations suivantes :

  • index : index incrémenté qui identifie la donnée
  • groupe : numéro du groupe auquel appartient la donnée
  • erreur : différence entre la direction corrigée par l'étalonnage et la direction moyenne du groupe (en degrés)
  • azimuth, pente et roulis : calculés depuis les valeur brutes sans correction d'étalonnage
  • éventuellement, les valeurs brutes des données d'étalonnage, c'est-à-dire les composantes de G et M dans la trame DistoX, sous forme de valeurs décimales ou hexagonales

Couleurs

Les données d'étalonnage sont affichées en jaune et un texte bleu, avec une alternance paire et impaire des numéros de groupes. Les données sans groupe sont grises.

Les couleurs de fond pour les donénes sont :

  • noir : valeurs normales
  • violet : données incomplètes (non utilisées pour l'étalonnage, mais utilisées pour le groupe)
  • marron : données saturées
  • gris : données supprimées (seulement si affichées)
  • rouge : données dont l'erreur est supérieure à 1°
  • vert : données différant de la valeur moyenne du groupe pour une valeur supérieure à la valeur distance de groupe

ACTIONS

  • Appui sur une donnée : ouvre la fenêtre d'édition

BOUTONS

  • Bascule le DistoX en mode étalonnage (cela peut échouer au premier essai)
  • Réinitialise la connexion BT ou contrôle de DistoX à distance
  • Télécharge les données d'étalonnage du DistoX
  • Affecte un numéro de groupe aux données, ou réinitialise les numéros de groupe
  • Calcule les coefficients d'étalonnage
  • Distribution spatiale des données d'étalonnage
  • Lit et affiche les coefficients d'étalonnage stockés dans le DistoX
  • Écrit les coefficients d'étalonnage dans le DistoX. Activé uniquement après le calcul des coefficients. Les coefficients peuvent également être écrits depuis la fenêtre de résultats de calculs. Un avertissement est affiché sr la couverture des directions est inférieure à 95% ou si l'erreur moyenne est supérieure à 0.5°
  • Ouvre la fenêtre de recherche

Pour le moment, seul le téléchargement de données "A la demande" est activé pour les données d'étalonnage. Les données téléchargées sont ajoutées à l'étalonnage actuel. Par conséquent, vous pouvez prendre quelques mesures, télécharger les données, vérifier si elles sont satisfaisantes, sinon en prendre d'autres et les télécharger de nouveau. Et cela, jusqu'à trouver satisfaction dans vos données d'étalonnage. Les données téléchargées peuvent être "désactivées" en définissant le numéro de groupe à "0" (zéro). Les données désactivées ne sont pas utilisées dans le calcul des coefficients d'étalonnage. Elles ne sont pas non plus supprimées (normalement, elles ne sont pas affichées dans la liste). Les données supprimées ne sont pas prises en compte dans les affectations des numéros de groupe et dans le calcul des coefficients.

Bluetooth

Ce bouton réinitialise la connexion Bluetooth de TopoDroid.
Il permet également à l'utilisateur de contrôler le DistoX à distance [A] : éteindre ou allumer le laser et prendre des visées pour l'étalonnage. Les données sont immédiatement téléchargées si le mode est défini sur "Continu".
Dans ce cas, il est également possible de prendre (et télécharger) un groupe de quatre visées d'étalonnage avec un court délai entre elles (controlé par le paramètre "attente laser" et "attente visée").

Le bouton Bluetooth est désactivé tant que TopoDroid prend des visées d'étalonnage.

Calculs de l'étalonnage

Le bouton "roue dentée" démarre le calcul des coefficients d'étalonnage.
Quand le calcul se termine, la fenêtre de coefficients est affichée.
S'il n'y a pas suffsamment de données ou de groupes de données, le calcul n'aboutit pas et un avertissement est affiché.
L'étalonnage échoue si la limite d'itération est atteinte avant la fin du calcul. Cela arrive généralement si les numéros de groupes sont affectés de manière erronnée. Dans cette situation, un message est également affiché.

Avertissements

  • Données saturées: données avec une valeur de champ magnétique anormale. Une cause possible est la présence d'un champ magnétique anormal. Les données saturées possèdent une couleur de fond marron.
  • Données nulles : données d'étalonnage reçues incorectement. La donnée ne possède pas de valeurs et par défaut n'est pas assignée à un nombre. Toutefois, elle est utilisée pour le calcul des groupes. Les données nulles son affichées avec un fond violet.

MENUS

  • Afficher/cacher : affiche ou non toutes les données, incluant celles supprimées
  • Valider : validation d'un étalonnage par rapport à un autre
  • Paramètres : identique à la fenêtre d'étalonnage
