[3] FENÊTRE DES PÉRIPHÉRIQUES

IMPORTANT Avant de vous lancer dans un relevé topographique, vous devez connecter et apairer votre appareil Android avec votre périphérique et l'étalonner. Appuyez sur le bouton Périphériques dans la Fenêtre principale afin d'ouvrir la Fenêtre des périphériques. La liste des périphériques connus est affichée. Les périphériques supportés sont les suivants : DistoX, DistoX2, DistoX-BLE, SAP5 et BRIC4 (ou 5).


S'il n'est pas apparié automatiquement, le code PIN du DistoX est "0000", quatre zéros.
S'il n'est pas apparié automatiquement, le code PIN du SAP5 est "000000", six zéros.

TopoDroid n'utilise qu'un seul appareil à la fois. Pour le sélectionner, appuyez sur son nom dans la liste. Le nom du périphérique sélectionné et affiché en hat de l'écran, sous les boutons. Si aucun appareil n'est sélectionné, un message rouge s'affiche : "Aucun périphérique sélectionné".

Si la liste des périphériques est vide, votre périphérique à besoin d'être appairé avec votre appareil Android. Cela peut être réalisé avec l'application Paramètres d'Android.

Appareils pris en charge
TopoDroid n'utilise que des appareils connectés et appairés avec le système Android. Les appareils sont sélectionnés dans la liste des appareils appariés avec Android en fonction de leur nom Bluetooth.
Les appareils suivants sont pris en charge ou en cours de prise en charge :

Nom du périphérique
Par défaut, TopoDroid utilisé l'adress MAC Bluetooth comme nom de périphérique. Les périphériques peuvent être identifiés par un alias pour les distinguer entre eux plus facilement.

Nota : Les boutons et menus marqués d'une [*] sont spécifique aux DistoX et nécessite que le périphérique soit allumé.

ACTIONS

  • appuyez sur la ligne d'un périphérique le sélectionne pour en faire l'appareil à utiliser.
  • Un appuis long sur un périphérique permet de définir sonalias ou de choisir un "second" DistoX

BOUTONS

  • Réinitialise la connexion Bluetooth.
  • Information du périphérique [N,*]
  • Fonctions mémoire des DistoX ou BRIC4 [*]
  • Active/désactive le mode d'étalonnage du DistoX [*]
  • Étalonnage des DistoX
  • Lit et affiche et les coefficients d'étalonnage depuis le DistoX [A,*]

[*] Ces actions nécessitent une communication avec l'appareil.

Pour le DistoX, tous les boutons sont actifs.
Pour le BRIC4, les trois premiers boutons sont actifs.
Pour le SAP5, seul le premier bouton est actif.

Si l'activation du mode d'étalonnage du DistoX échoue au premier essai, répétez l'opération une seconde fois.

La lecture du périphérique échoue si TopoDroid est connecté au périphérique et télécharge des données. Cela peut arriver quand la Fenêtre périphériques est ouverte depuis la Fenêtre des visées.

MENUS

  • Déconnecte déconnecte le périphérique actif [A]
  • Télécharge des firmwares vers le DistoX (v. 2 and BLE) [E,*]
  • Affiche les journaux des paquets du DistoX [T]
  • Paramètres
  • Aide

PARAMÈTRES DU PÉRIPHÉRIQUE

  • Bluetooth: Activation/désactivation et vérification au démarrage [défaut "vérifier"]
  • Nombre de nouvelles données: récupère le nombre de nouvelles donénes avant de les télécharger (mode à-la-demande) [défaut "non"]

Les paramètres suivants ne s'appliquent qu'au DistoX (qui utilise une connexion Bluetooth classique).

  • Mode de connexion. "A-la-demande", "continue", or "multiple" [défaut "A-la-demande"]
  • Type de Socket DistoX. Normal ou "Non sécurisé" [defaut "Non sécurisé"]
  • DistoX Contournement Z6: un erreur de dépassement de délai survient sur les périphériques Z6 et potentiellement d'autres. Il est recommandé de laisser cette option activée [défault "activé"]

Mode de connexion
Il existe trois modes de connexion pour les DistoX :

  • Mode À-la-demande, les données sont téléchargées en rafale et quand il n'y à plus aucune données à télécharger, la connexion est fermée.
  • Mode continu, TopoDroid reste connecté au DistoX et les visées sont téléchargées dès leur prise. Si le DistoX ou l'apareil Android se déconnecter, TopoDroid tente de se reconnecter. Sinon, la connexion Bluetooth est fermée.
  • Mode multiple, ce mode resemble au mode "A-la-demande" mais un appuie long sur le bouton "Télécharger" dans le fenêtre de visées ou de dessin change de DistoX. Si vous avez défini un "second" DistoX (dans la fenêtre Périphériques), il est changé par le DistoX actuel. Autrement, il vous est proposé une liste DistoX appairés dans laquelle vous pouvez choisir le bon.

Avec l'option "Multiple", le DistoX courant est affiché dans le titre de la fenêtre visées et Dessin.
Le mode de connexion ne s'applique uniquement au téléchargements des données. Les autres fonctions utilisent un mode d'intérogation unique.

Les SAP5 and BRIC4 sont toujours connectés en mode continu.

Paramètres personnalisés: Délai de connexion, Second DistoX, Data pause, Data-ready wait, Laser pause, Shot pause.
