DÉPANNAGE DU DISTOX

Appairage de l'appareil
Avant que le TopoDroid puisse se connecter à un appareil, celui-ci doit être appairé avec l'Android.
Vous pouvez appairer Android avec le DistoX actif via le menu d'appairage ou vous pouvez utiliser l'application Paramètres pour appairer le DistoX.

Si Android demande le code PIN du DistoX, entrez 0000 (quatre zéros).
Pour le code PIN SAP5, entrez 000000 (six zéros).
Android ne devrait pas demander le PIN BRIC4, ni le PIN DistoXBLE.

Une fois jumelé avec le DistoX, l'Android doit conserver le jumelage et ne plus demander le code PIN. S'il demande le code PIN à chaque fois que vous téléchargez des données depuis le DistoX, essayez d'utiliser le paramètre de socket non sécurisé.

Si vous avez des problèmes de connexion et de téléchargement de données avec un appareil DistoX, essayez d'activer le paramètre de contournement du Z6.

Le DistoXBLE se connecte mais ne télécharge pas
Cela peut se produire immédiatement après avoir activé le Bluetooth Android. La solution consiste à éteindre et à rallumer le DistoXBLE. TopoDroid devrait reprendre la connexion automatiquement et commencer à télécharger les données.
