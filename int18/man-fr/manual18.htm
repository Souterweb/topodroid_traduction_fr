FAIRE UN CROQUIS EN 3D

L'esquisse 3D se fait autour d'une visée de cheminement à la fois. L'affichage est donc centré sur la visée sélectionnée.
La fenêtre d'esquisse 3D comporte trois modes (et trois barres de boutons différentes) :

  • déplacer
  • dessiner
  • sélectionner (uniquement pour la vue "cheminement")

Comme pour la fenêtre de dessin 2D, les boutons verts les plus à gauche indiquent le mode actuel.
Les autres boutons du mode déplacement sont les suivants :

  • mode d'affichage
  • notes
  • parois
  • rafraîchir

En mode dessin, vous pouvez tracer des lignes de contour. Les lignes de contour de la vue du "cheminement" se situent dans le plan 3D plongeant avec ce cheminement. Celles des vues en coupe se situent dans le plan de coupe.

BOUTONS :

  • annuler
  • rétablir

Le mode effacement dispose de boutons :

  • défaire
  • rétablir
  • taille de la gomme

Pour le mode sélection :

  • mode de sélection : soit des points de contour/section, soit des stations voisines
  • taille de la sélection
  • effacer
  • ouvrir : soit la section, soit la vue "cheminement" voisine
  • effacer la sélection


La vue "cheminement" contient également les visées de cheminement connectées. Le fait d'appuyer sur l'une des stations voisines permet de déplacer la vue de "cheminement" vers cette autre station.

Une section peut être créée lorsque deux points sur deux lignes de contour ont été sélectionnés. Le plan de la section peut être "vertical" ou "horizontal". S'il est vertical, il contient les deux points sélectionnés et l'axe Y est la verticale. S'il est horizontal, il passe par leur point médian et l'axe Y est l'azimut de la jambe. Les deux points (ou leurs projections) sont affichés à l'écran.
Un menu permet de commuter entre les choix "vertical" et "horizontal".Lorsque le canevas affiche une section x, le bouton "sélectionner" se transforme en bouton "retour".


Un appui court ferme la section transverse et ramène à la vue de cheminement .Une section transverse peut être rouverte en sélectionnant le point de la section transverse (à mi-chemin entre les deux points de base de la section transverse) et en appuyant sur le bouton "ouvrir".
