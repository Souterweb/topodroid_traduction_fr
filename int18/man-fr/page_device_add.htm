BOÎTE DE DIALOGUE AJOUT DE PÉRIPHÉRIQUE

Cette boîte de dialogue permet à l'utilisateur d'ajouter un appareil à la liste des appareils périphériques à TopoDoid. La boîte de dialogue comporte les champs suivants :

  • Adresse
  • Modèle et numéro
  • Alias (optionnel)


L'adresse est sélectionnée à partir d'une liste déroulante qui montre les adresses MAC des appareils couplés qui ne sont pas dans la base de données TopoDroid.
L'adresse peut également provenir des détails de l'appareil connecté dans les paramètres de l'application.

Le modèle est sélectionné dans une liste déroulante des appareils pris en charge.

Le numéro doit être un numéro à quatre chiffres. Il peut comporter des zéros initiaux.
Le numéro de l'appareil peut être lu à partir des informations sur l'appareil lui-même.

L'alias est facultatif et peut être laissé vide.
