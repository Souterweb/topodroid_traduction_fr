MODIFIER UN ÉLÉMENT POINT DU CROQUIS
Boîte de dialogue

Cette boîte de dialogue s'ouvre lorsque vous sélectionnez un symbole de "Point" puis appuyez sur le bouton Propriétés de l'élément (dans la fenêtre d'Esquisse en mode "Modifier").

Propriétés du Point :

  • Taille : au choix parmi XS, S, M, L, XL
  • Orientation : si le point est orientable
  • Calques : les vues du fond sur lesquels le point est affiché, disponibles lorsque l'option "Calques" est réglée sur "par élément" [T].
  • Texte : un commentaire pour les points de type "étiquette"
  • Options génériques Therion

Les couches du fond sont présentées sous forme de code alphabétique avec des cases à cocher :

  • B : couche de base (par défaut)
  • F : couche du sol (F pour Floor)
  • D : couche du décor (D pour Decoration)
  • C : couche du plafond (C pour Ceiling)
  • A : couche des objets (A pour Artefacts)

Vous pouvez modifier les vues auxquelles le point appartient. L'icône du point n'est pas affichée si toutes ses vues sont désactivées. Le point sélectionnable sera néanmoins affiché en mode édition, de sorte que la boîte de dialogue d'édition du point reste accessible.

Annuler ferme la boîte de dialogue.
Enregistrer confirme les modifications et ferme la boîte de dialogue.

Lorsque la fonction de déplacement dans les coins est activée, l'orientation du point peut également être modifiée en glissant vers le bas à partir des coins supérieurs du canevas (ce qui fait tourner l'orientation du point dans le sens inverse des aiguilles d'une montre et dans le sens des aiguilles d'une montre, respectivement).

La touche RETOUR ferme la boîte de dialogue sans enregistrer les modifications.
