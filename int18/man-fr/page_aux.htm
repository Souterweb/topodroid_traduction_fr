APPLICATIONS AUXILIAIRES À TopoDroid

Lorsque vous travaillez avec TopoDroid, certaines applications complémentaires et auxiliaires peuvent vous être utiles :

  • Proj4 est une application de conversion de coordonnées (très simple), utilisée par TopoDroid pour convertir des valeurs de latitudes et longitudes en d'autres systèmes de coordonnées.
  • Une application GNSS. Vous pouvez l'utiliser pour prendre les coordonnées GPS d'un point et les importer dans TopoDroid.
  • CrowdMag est une application du NOAA National Geophysical Data Center, utile pour obtenir des informations sur l'état actuel du champ magnétique terrestre.
  • Une application pour prendre des photos (généralement il y a une application Camera).
  • SQLiteViewer (ou toute autre application de gestion SQLite) peut être utile pour inspecter la base de données TopoDroid.
  • un gestionnaire de fichiers correct tel que MiXplorer, pour examiner les fichiers et faire des copies de sauvegarde.
  • SQLiteViewer (ou toute autre application de gestion SQLite) peut être utile pour inspecter les bases de données TopoDroid.


Applications GNSS

  • GPSTest est considéré comme "l'application GNSS" de référence. TopoDroid lit les coordonnées à partir d'un fichier journal de GPSTest. Les coordonnées peuvent également être stockées dans le presse-papiers et collées dans la boîte de dialogue d'insertion de points géolocalisés de TopoDroid.
  • GPX Recorder et GPS Position : TopoDroid lit les traces enregistrées (fichiers gpx et csv, respectivement) et utilise les coordonnées du dernier point.
  • GPS Point : TopoDroid lit les coordonnées exportées dans un fichier de points (en texte clair).
  • MobileTopographer : TopoDroid lit les coordonnées du fichier de points dans la liste de points de MobileTopographer. Malheureusement, cette application n'est plus gratuite.

En outre, selon la façon dont vous traitez vos relevés topos, vous voudrez peut-être installer :

  • une visionneuse / éditeur d'images (généralement, il y a une visionneuse d'image préinstallée)
  • un visualiseur / éditeur DXF
  • un visualiseur / éditeur de feuille de calcul
  • une visionneuse SVG
  • un archiveur ZIP
  • GoogleEarth
  • une visionneuse de fichier de traces gpx

 

Proj4
TopoDroid stocke les coordonnées des points dans le système de référence géographique long-lat WGS-84. Les altitudes sont stockées par rapport à l'ellipsoïde et au géoïde.

L'application Proj4 est utilisée par TopoDroid de deux manières pour transformer les coordonnées entre le WGS-84 et d'autres systèmes de coordonnées :

  • pour convertir les coordonnées d'un point long-lat (WGS-84) vers un autre système de référence de coordonnées.
  • pour saisir les coordonnées d'un point dans un système de coordonnées différent.

La syntaxe proj pour les systèmes de référence de coordonnées peut être trouvée sur spatialreference.org.
