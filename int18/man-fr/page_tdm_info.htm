GESTIONNAIRE DE PROJET - POINTS GÉOLOCALISÉS

Cette boîte de dialogue est ouverte par le bouton GNSS (Géolocalisation et Navigation par un Système de Satellites) de la fenêtre du projet topographiques.
La boîte de dialogue liste les informations de géolocalisation des relevés du projet :

  • nom
  • station
  • coordonnées WGS84
  • système de coordonnées supplémentaire
  • coordonnées dans le système utilisé

Les relevés sans géolocalisation ne sont pas listés.
