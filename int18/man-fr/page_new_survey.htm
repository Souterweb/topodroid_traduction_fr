NOUVEAU RELEVÉ TOPOGRAPHIQUE

Cette boîte de dialogue s'ouvre en appuyant sur le bouton Ajouter  de la Fenêtre principale.

Les informations sur le nouveau relevé sont :

  • Nom
  • Date
  • Équipe
  • Station initiale
  • Déclinaison magnétique
  • Description du relevé topo
  • Sections transverses privées (ou partagées)
  • Mode de données plongée [T]

Le nom du relevé est obligatoire.
Il doit être unique dans la base de données. Les noms des relevés topographiques sont utilisés pour répertorier les relevés topographiques dans la fenêtre principale et TopoDroid distingue les relevés topographiques par leur nom (bien que chaque relevé ait une identification numérique unique dans la base de données).

Attention. Le nom du relevé ne peut pas contenir d'espaces, ni les caractères "/" (barre oblique, qui est le séparateur de nom de chemin), " :" (deux points), "*" (étoile) et "\" (barre oblique inverse). Les barres obliques inverses sont supprimées. Les espaces sont remplacés par "_" (trait de soulignement), les barres obliques et les deux-points par des tirets "-", et les étoiles par "+" (plus).

La date est écrite au format "aaaa.mm.jj". Par exemple le 6 janvier 2015 devient "2015.01.06". Il est prédéfini à la date actuelle.

L'équipe et la description sont vides (sauf si vous avez défini une équipe par défaut dans les paramètres de l'application). Ils peuvent être laissés vides en toute sécurité, bien qu'il soit préférable de prendre l'habitude de les remplir.

La déclinaison magnétique est de 0,0 par défaut. Si vous le connaissez, vous pouvez la saisir ici, sinon vous pouvez le rechercher sur le Web après avoir trouvé les coordonnées d'un point fixe (par exemple l'entrée de la grotte) avec le GPS [A].

Les sections transverses de la station "Vers" peuvent être soit partagées entre les deux esquisses topographiques (PLAN et COUPE), soit spécifiques à chacune des esquisses (privées). Ce choix doit être fait lors de la création du relevé topographique et ne peut être modifié ultérieurement car il affecte les croquis existants.

Boutons :

ENREGISTRER : le nom du nouveau relevé topographique est ajouté à la base de données.
Ouvrir : enregistre et ouvre immédiatement la fenêtre de données du relevé topographique.
Annuler : abandonne et ferme la fenêtre

Mode de données "plongée"
Le mode de données "plongée" est une fonction expérimentale. Les données ne peuvent être saisies que manuellement, et dans le format suivant :

  • Station DEPUIS
  • Station VERS
  • Profondeur à la station DEPUIS
  • Azimut de la direction DEPUIS > VERS
  • Distance entre la station DEPUIS et la station VERS
  • GDHB (LRUD) à la station DEPUIS (facultatif)

Les données des visées arrière ne sont pas prises en charge en mode de données "plongée".
