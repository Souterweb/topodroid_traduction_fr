SKETCH SECTION POINT

This dialog is opened when you select a section-point item and tap the Edit button in the Drawing window (in "edit" mode).

Point properties:

  • size (XS, S, M, L, XL)
  • layers, the canvas views on which the point is displayed. This property is only available if the Custom Settings > Sketching > Layers setting is set to by item [T]
  • Therion options: the sketch point has always the "-scrap" option followed by the name of the x-section
  • x-section checkbox to display the xsection outline in the sketch

Canvas views (layers)
The canvas views are presented as checkboxes with the view letter-code. You can change the views the point belongs to.

The point icon is not displayed at all if all its views are OFF. In this case tthe selectable point will nevertheless be shown in edit mode, so that the point edit dialog remains accessible, and a layer can be assigned to the point.

The Draw button to open the x-section sketch, if it exists. Changes to the xsections sketch are not propagated to the outline in the sketch: to update it, close the outline and open it again.

Save confirms the changes and closes the dialog.
Cancel closes the dialog.
The BACK key closes the dialog without saving the changes.
