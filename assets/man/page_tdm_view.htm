Project Plan View

The Project Plan View window is presented when you tap the Plan button in the Project Manager window.
It displays selected surveys of the cave project on the horizontal plane (thus in plan view), with midline and station names.

Each survey has a different color. After a while the colors repeat therefore you should display only two or three surveys at a time.
The display can be panned and zoomed in/out.

By selecting a station of one survey (tapping on its name) that survey can be dragged around while keeping the others fixed.
By this means the station can be brought close to a station of another survey.
The two stations can be selected at the same time with a tap nearby, and an "equate" between the two stations added to the cave project.

Equates are shown as dashed red lines, and equated stations have a round background. The size of the station names is equal to that in the Drawing window.

Equates can be also added manually through the Equate Dialog. The list of equates is shown with the Equates Dialog, through which they can be removed from the cave project.

Menus

  • Add equate: opens a dialog where the user can set which stations are to be equated. In order to manually add an equate between surveys of the cave project, tap the "OK" button".
  • List equates between surveys of the cave project.
  • Reduce number of stations identified
  • Increase number of stations identified

Equates list

On the dialog presented when clicking on the List equates button you can drop an equate by tapping its entry in the dialog listing.
