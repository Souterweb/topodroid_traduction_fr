Add-device dialog

With this dialog the user can add a device to the TopoDroid device list. The dialog has the fields
  • Address
  • Model and number
  • Nickname (optional)

The address is selected from a drop-down list which shows the MAC-addresses of the paired devices that are not in the TopoDroid database.
The address can also be checked from the details of the connected device in the Settings app.

The model is selected from a drop-down list of supported devices.

The number must be a four-digit number. It can have leading zeros. The number of the device can be read from the info on the device itself.

The nickname is optional and can be left blank.
