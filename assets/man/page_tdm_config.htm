PROJECT MANAGER - Project window

The cave Project window is opened tapping a project name in the listing of the Project Manager window.

It displays the list of surveys that comprise the project. The list is empty if the project has no surveys.
Each survey can be selected for further actions by tapping its checkbox.

Menus

  • Close the window
  • Export the cave project (Therion or Survex)
  • Delete the cave project
  • Help

Buttons

  • insert surveys
  • remove the selected surveys
  • 2D display of the selected surveys, in plan view
  • display the survey geo-points
  • list the project equates
  • 3d view of the project
The insert button display the list of surveys in the TopoDroid database. Surveys are added to the project by tapping their checkboxes and confirming with the OK button.

Project 3D view

The project is displays in the 3D viewer beginning with the first survey.
The following surveys are shown in the 3D only if they are (directly or indirectly) connected to the first survey.

If several surveys are georeferenced, they can all be displayed in 3D, together with the connected surveys.
